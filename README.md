# README

Simple location app for share the user location.

Requirements:

* Ruby 2.7.2

* Rails 6.1

* Postgresql

* Devise

* Open Layers

## Usage

```git
git clone git@github.com:malikahsan/location-app.git
cd location-app
set your own username and password on database.yml
bundle install
# database setup
rails db:setup
rails db:seed
# run app
rails s
```
